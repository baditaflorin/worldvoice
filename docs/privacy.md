# Privacy

WorldVoice is designed so microphone audio stays on the user's device.

## Collected Data

No analytics are collected in v1.

No microphone audio is uploaded.

No accounts are used.

No cookies are intentionally set by the app.

## Local Storage

WorldVoice stores only local preferences:

- Selected preset.
- Intensity, space, and color controls.
- Battery-saver mode.

These values are stored in IndexedDB, with a narrow `localStorage` fallback if IndexedDB is unavailable.

## Network Requests

The app fetches static assets from GitHub Pages:

https://baditaflorin.github.io/worldvoice/

Future external model packs must be public, secret-free assets and must be documented in the model manifest.

## Opt Out

Clear the browser's site data for `baditaflorin.github.io` to remove local settings and cached assets.
